🧠 App Name: Memory Card Game

📌 Description:
This is a fun and interactive memory matching game built using HTML, CSS, and JavaScript. Players flip cards to find matching pairs.

🚀 Features:

  • 🎮 Multiple difficulty levels
  • ⏱ Timer system
  • 🎯 Move counter
  • ⭐ Star rating
  • 🏆 Leaderboard (localStorage)
  • 🎴 3D card flip animation
  • 🔊 Sound effects
  • 📱 Responsive design
