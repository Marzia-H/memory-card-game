📘 About This App

App Name: Memory Card Game

Description:
This is a simple memory matching game built using HTML, CSS, and JavaScript. The goal is to match all pairs of cards by flipping them.

Features:

  • Flip cards to reveal symbols
  • Match pairs to win
  • Restart game anytime
  • Simple and interactive UI

How to Use:

  1. Click on a card to flip it
  2. Flip another card to find a match
  3. If matched, cards stay open
  4. If not, they flip back
  5. Match all cards to win

Technologies Used:
HTML, CSS, JavaScript
